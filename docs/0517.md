# Java TreeMap 类

> 原文:[https://www.javatpoint.com/java-treemap](https://www.javatpoint.com/java-treemap)

![Java TreeMap class hierarchy](../Images/66c0f62c6462c439b0c978f9f84aee74.png)

Java TreeMap 类是一个基于红黑树的实现。它提供了一种以排序顺序存储键值对的有效方法。

关于 Java TreeMap 类的要点是:

*   Java TreeMap 包含基于键的值。它实现了导航映射接口并扩展了抽象映射类。
*   Java TreeMap 只包含唯一的元素。
*   Java TreeMap 不能有空键，但可以有多个空值。
*   Java TreeMap 是非同步的。
*   Java TreeMap 保持升序。

class TreeMap1{
 public static void main(String args[]){
   TreeMap map=new TreeMap<integer>();  
	  map.put(100,"Amit");  
	  map.put(102,"Ravi");  
	  map.put(101,"Vijay");  
	  map.put(103,"Rahul");  

	  for(Map.Entry m:map.entrySet()){  
	   System.out.println(m.getKey()+" "+m.getValue());  
	  }  
 }
}</integer>

### HashMap 和 TreeMap 有什么区别？

| 模拟 | 树图 |
| 1) HashMap 可以包含一个空键。 | TreeMap 不能包含任何空键。 |
| 2) HashMap 不维护任何顺序。 | TreeMap 保持升序。 |
