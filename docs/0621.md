javax.servlet 和 javax.servlet.http 包代表 servlet api 的接口和类。

**javax.servlet** 包包含许多由 servlet 或 web 容器使用的接口和类。这些并不特定于任何协议。

**javax.servlet.http** 包包含只负责 http 请求的接口和类。

让我们看看 javax.servlet 包有哪些接口。
