**ServletOutputStream** 类提供了一个将二进制数据写入响应的流。这是一个抽象类。

**ServletResponse** 接口的 **getOutputStream()** 方法返回 ServletOutputStream 类的实例。它可以被理解为:

```
ServletOutputStream out=response.getOutputStream();

```
