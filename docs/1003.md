## RESTful 网络服务的优势

**快** : RESTful Web 服务之所以快，是因为没有像 SOAP 那样严格的规范。它消耗更少的带宽和资源。

**语言和平台无关** : RESTful web 服务可以用任何编程语言编写，也可以在任何平台上执行。

**可以使用 SOAP** : RESTful web 服务可以使用 SOAP web 服务作为实现。

**允许不同的数据格式** : RESTful web 服务允许不同的数据格式，如纯文本、HTML、XML 和 JSON。
