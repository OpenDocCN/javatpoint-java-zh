# 示例:打印一个网址的图像

> 原文:[https://www . javatpoint . com/js oup-example-print-images-of-an-URL](https://www.javatpoint.com/jsoup-example-print-images-of-an-url)

在这个例子中，我们将打印一个网址的全部图像。为此，我们调用 select()方法传递“img[src~=(？i)\\。(png|jpe？g|gif)]作为参数，以便它可以打印 png、jpeg 或 gif 图像。

```
import org.jsoup.Jsoup;
import org.jsoup.nodes.Document;
import org.jsoup.nodes.Element;
import org.jsoup.select.Elements;
public class JsoupPrintImages {
	 public static void main( String[] args ) throws IOException{
			Document doc = Jsoup.connect("http://www.javatpoint.com").get();
			Elements images = doc.select("img[src~=(?i)\\.(png|jpe?g|gif)]");
			for (Element image : images) {
				System.out.println("src : " + image.attr("src"));
				System.out.println("height : " + image.attr("height"));
				System.out.println("width : " + image.attr("width"));
				System.out.println("alt : " + image.attr("alt"));
			}

}
}

```
